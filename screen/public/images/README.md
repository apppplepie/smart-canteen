# 共享图片目录说明

前端与接口共用的图片根目录，访问路径为 `/images/...`（Vite 将 `public` 映射到根路径）。

## 目录用途

| 目录 | 用途 | 建议尺寸/比例 |
|------|------|----------------|
| `dishes/` | 菜品主图 | 建议 600×800 或 3:4，缩略图放 `thumbnails/dishes/` |
| `windows/` | 窗口头图/主打菜图 | 建议 400×300 或 4:3 |
| `zones/` | 食堂区域（Bento）轮播图 | 建议 800×600 或 4:3 |
| `lost-found/` | 失物招领/寻物相关图 | 建议 800×600 |
| `thumbnails/` | 各业务缩略图，子目录与上面一致 | 与主图同比例，宽度建议 200–400px |

尺寸与缩略图路径由后端表 `image_asset` 维护，上传或生成后写入该表。
